Unreal Weapons System

When I started learning Unreal, my weapons system was one of my first mini-projects. It started as a pet project to both practice what I was learning in class and document functions and features for reference in other projects. Each class that involves Unreal has since contributed to the weapons system in some way. At this point it can do what most FPS games feature. Weapon holstering, switching, fire modes, pick up, drop, mag size, ammunition type, fire rate, etc. Each attribute can be customized by the developer to create unique combinations of weapons. An AR that has both burst and single shot with a 300 RPM and high explosive ammo. A Sniper rifle that is bolt action with armor piercing rounds.
